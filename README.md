# CLI-сервис сокращения ссылок

Консольное Java-приложение для создания и управления короткими ссылками с ограничением по количеству переходов и временем жизни (TTL). Подходит для личного использования или демонстрации навыков ООП.

## Функционал

- Сокращение длинных URL в уникальные короткие ссылки.
- Ограничение по количеству переходов (лимит).
- Автоматическое удаление ссылок по истечении времени жизни (TTL).
- Уникальная идентификация пользователя через UUID при первом использовании.
- Защита доступа: только владелец может управлять своими ссылками.
- Переход по короткой ссылке с открытием в браузере.
- Работа в мультипользовательском режиме.
- Данные хранятся в памяти и сохраняются между запусками.
- Оповещения о превышении лимита или окончании срока действия.

## Как запустить

### Требования
- Java 17+ (например, OpenJDK)
- IntelliJ IDEA (опционально)

### Запуск 
### IntelliJ IDEA
Открой проект в IntelliJ IDEA.
Нажми на класс Main.java.
Нажми зелёную стрелку → Run 'Main.main()'
### Командная строка
```bash
java -cp out/production/ShortLinkService Main
```
## Команды
### create url limit - Создать короткую ссылку с лимитом переходов
### go shortLink - Перейти по ссылке (откроется в браузере)
### list - Показать все мои ссылки
### delete shortLink - Удалить свою ссылку
### help - Показать справку
### exit - Выйти из программы
